навигация в Web для людей с 
ограниченными возможностями
Создать 
бесплатное и доступное 
фоновое расширение для браузера,
позволяющее осуществлять
навигацию и управление
на любых веб страницах  
с помощью движений лица 
и голосовых команд.
Цель проекта
Существующие технологии
  • Дорого

  • Требуют спец. оборудования

  • Не для всех
Предлагаемое решение 
  • Бесплатно и устанавливается в 2 клика.

  • Использует только встроенные
    камеру и микрофон.

  • Индивидуальная настройка и адаптация.

  • Базируется только на новейших web API: 
    webGl, webAudio,  webRTC, speechAPI...

  • Open Source.
Почему это важно:
Web - важнейшая социальная среда
с возможностью общения, 
самовыражения, обучения.

Это возможность 
быть равноправным, понятым
и оцененным.

  7. Интерфейс обучения системы.
